# Schedule for Tsoding Streams

Deployed in https://tsoding.github.io/schedule-beta/

## Quick Start

```console
$ npm install
$ make
$ <browser> dist/index.html
$ make -j2 watch                  # for watch mode
                                  # -j2 is required to run scss and ts watchers in parallel
```
